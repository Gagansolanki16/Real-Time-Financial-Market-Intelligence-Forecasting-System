# Real-Time-Financial-Market-Intelligence-Forecasting-System
This repository contains a comprehensive end-to-end data pipeline designed to analyze and predict stock market trends. The project integrates automated data extraction, machine learning for predictive modeling, and interactive business intelligence for real-time decision-making.

🚀 Project ArchitectureThe system follows a modular pipeline to ensure data integrity and real-time performance:Data Acquisition: Python scripts fetch live market data from APIs (Yahoo Finance).Storage: Processed data is stored in a SQL Database for historical tracking and consistency.Machine Learning: An XGBoost model processes historical trends to generate future price forecasts.Web Interface: A Streamlit app provides a user-friendly way to interact with the model and view live predictions.Business Intelligence: Power BI connects to the database for deep-dive exploratory data analysis (EDA).🛠️ Tech StackBackend: Python (Pandas, NumPy, Scikit-learn)Machine Learning: XGBoost RegressorFrontend/Web: StreamlitDatabase: SQLVisualization: Power BI Desktop✨ Key FeaturesLive Market Pulse: Real-time data fetching and visualization.Predictive Modeling: Advanced forecasting using Gradient Boosting to identify potential market trends.Interactive UI: Users can select different stock tickers and timeframes through the Streamlit interface.Institutional Reporting: Power BI dashboards for analyzing volatility, moving averages, and model error metrics ($RMSE$, $MAE$).Run the Web AppLaunch the Streamlit interface:Bashstreamlit run app.py
